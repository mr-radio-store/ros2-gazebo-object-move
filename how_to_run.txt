Control ROS2 with arrow key 

How to run ROS2
1 Access file & initial build
ccd ~/Desktop/ROS2_Gazebo_Obj_Move/robot_ws
rm -rf build/my_robot_harmonic install/my_robot_harmonic
colcon build --packages-select my_robot_harmonic --symlink-install

2 Verify the build succeeded
source install/setup.bash
ros2 pkg executables my_robot_harmonic
>>>You should see
my_robot_harmonic teleop_harmonic

3 launch
ros2 launch my_robot_harmonic simulation_harmonic.launch.py


Second terminal
cd ~/Desktop/ROS2_Gazebo_Obj_Move/robot_ws
source install/setup.bash
ros2 run my_robot_harmonic teleop_harmonic
